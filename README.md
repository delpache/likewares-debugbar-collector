# Package Laravel Debugbar collector pour Likewares CRM

Ce paquet a pour but de fournir des informations supplémentaires sur l'installation d'Akaunting, les versions et les détails du serveur.

## Installation

Vous pouvez installer le paquet via composer :

```bash
composer require likewares/laravel-debugbar-collector
```
